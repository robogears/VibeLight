# What's new in v0.1.5

## "Move to Applications" on first launch
- If you run VibeLight from Downloads (or a quarantined copy), it now offers to
  **move itself into your Applications folder** — one click, styled like the
  rest of the app. This keeps it tidy and, importantly, lets the self-updater
  install future versions seamlessly.

## Tiny test build
- Mostly here to prove the updater: if you're on v0.1.4, this should have
  arrived as **Update Now → download → Restart Now**, no manual download. 🎉

# Install / update

- **On v0.1.4 or later?** Click **Update Now** when offered (or Settings → About
  → Software Update), then **Restart Now**. No download needed.
- **Fresh install (macOS, Apple Silicon):** download `VibeLight-0.1.5-arm64.zip`,
  unzip, open it — and just say **Move to Applications** when it asks. First
  launch on macOS 15+: right-click → Open, or **System Settings → Privacy &
  Security → Open Anyway** (the app is not notarized).
- On first stream, macOS may ask to allow **VibeLight Stream** on the local
  network — click Allow.

## Requirements

- Apple Silicon Mac, macOS 15 or later.
- [Moonlight](https://moonlight-stream.org) installed and already paired with
  your host once (VibeLight imports that pairing).
- A Sunshine-family host (Sunshine / Apollo / Vibepollo) on the gaming PC.
